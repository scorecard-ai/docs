## Get Access to the Scorecard API

To gain access to the Scorecard API, create an Scorecard account by signing up [here](https://accounts.getscorecard.ai/sign-up?redirect_url=https%3A%2F%2Fapp.getscorecard.ai%2F) via GitHub/Google Single Sign-On or via email. After you have completed the signup process and acknowledged our terms, you will find the `SCORECARD_API_KEY` in the [settings](https://app.getscorecard.ai/settings).
