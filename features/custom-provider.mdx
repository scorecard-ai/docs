## Overview

Scorecard's AI proxy infrastructure provides seamless integration with all major AI providers, enabling you to configure multiple providers, switch between models effortlessly, and maintain consistent API calls across different services.

<Note>
Configure your provider API keys in [organization settings](https://app.scorecard.io/settings) to get started with any of the supported providers below.
</Note>

## Key Benefits

<AccordionGroup>
  <Accordion title="Easy Provider Management">
    Configure multiple AI providers from a single interface using standardized API key management. No need to handle different authentication methods or API formats.
  </Accordion>
  
  <Accordion title="Seamless Model Switching">
    Switch between different AI models and providers without changing your implementation. Test and compare models effortlessly to find the best fit for your use case.
  </Accordion>
  
  <Accordion title="Unified Integration">
    Use consistent API calls regardless of the underlying AI provider through our proxy layer. Your code remains clean and provider-agnostic.
  </Accordion>
  
  <Accordion title="Cost Optimization">
    Compare costs across providers and models to optimize your AI spend. Easily switch to more cost-effective options as they become available.
  </Accordion>
</AccordionGroup>

## Getting Started

<Steps>
  <Step title="Add Provider Credentials">
    Navigate to your [organization settings](https://app.scorecard.io/settings) and add API keys for your desired providers.
  </Step>
  
  <Step title="Create or Update Systems">
    When creating [prompts](/features/prompts) or [systems](/features/systems), select from any of the available models across all configured providers.
  </Step>
  
  <Step title="Test and Compare">
    Use Scorecard's [A/B comparison features](/features/a-b-comparison) to evaluate performance across different providers and models.
  </Step>
</Steps>
