## Overview

The Playground lets you test your AI agent against testcases and see results in real time. It has three main areas:

1. **Testset Panel** (left): Select a testset and view individual testcases
2. **Prompt Editor** (center): Configure your prompt template and model settings
3. **Results Panel** (right): View outputs and evaluation metrics

## Quick Start

1. **Select a Testset**: Use the dropdown to choose a testset with your test data
2. **Configure the Prompt**: Edit the prompt template using [Jinja syntax](https://jinja.palletsprojects.com/) with variables like `{{context}}` or `{{allInputs}}`
3. **Adjust Model Settings**: Switch to the Model Settings tab to configure temperature, max tokens, etc.
4. **Run**: Click **Try** to test on selected testcases, or **Kickoff Run** to run a full evaluation

## Prompt Templates

The prompt editor supports Jinja templating for dynamic content:

```jinja
You are a helpful assistant.

User question: {{question}}
Context: {{context}}
```

Use `{{allInputs}}` to include all input fields from your testcase.
