Welcome to the Scorecard Cookbook. Here you'll find code and guides for accomplishing common tasks with the Scorecard API and SDKs. To run these examples, you'll need an Scorecard account and associated API key [(create an account here)](https://app.getscorecard.ai/).

Most code examples are written in Python, though the concepts can be applied in any language.
