This page walks through how to create a [Testset](/features/testset-management) using the Scorecard UI.

# Create a Testset

1. Navigate to the `Testsets` section in your project dashboard.
1. Click on "New Testset".
1. Provide a name and optional description for the Testset.
1. Select the `Upload CSV or JSON` option.
1. In the next page, upload your Testset file.

<Callout intent = "warn">
Only Testsets with the default schema can be created via UI. If you want to create a Testset with a custom schema, you will have to do it via the SDK for now.
</Callout>

# Modify a Testset

1. Navigate to the Testset you want to modify in the `Testsets` section.
1. You can modify existing Testcases directly in the table.
1. Or you can also add new Testcases by clicking on the "Add Testcase" button.
